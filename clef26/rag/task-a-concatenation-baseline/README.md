# Simple Concatenation Baseline for Task4@LongEval'26

This directory contains a simple concatenation baseline for the RAG task of LongEval 2026 using the [LongEval ir_datasets extension](https://github.com/clef-longeval/ir-datasets-longeval).

To create a run, please run:

```
./baseline.py --output responses.jsonl
```

the format is in the TREC-RAG format: https://github.com/castorini/ragnarok/blob/main/docs/rag25.md
